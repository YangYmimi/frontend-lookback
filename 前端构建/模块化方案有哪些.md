### 前端模块化方案

#### CommonJS

* 同步加载依赖的模块
* 可复用于 Node 环境 ，例如做同构应用
* 成熟的第三方模块社区
* 无法直接运行于浏览器环境，必须通过工具转换为标准的 ES5

#### AMD

* 异步加载依赖的模块
* 并行加载多个模块
* 可在不转换代码的情况下直接在浏览器运行
* 可运行在浏览器和 Node 环境
* 代表性实践 `requirejs`

#### ES6 Module

* 语言层面上实现模块化
* 需转换成 ES5 实现

#### Module in CSS

* `@import` 语句
